# Estructura de carpetas

```
web-app-mobile/
├── public/              # archivos estáticos
├── src/
│   ├── assets/          # imágenes, íconos y recursos importables
│   ├── components/      # componentes reutilizables de UI
│   ├── interfaces/      # tipos e interfaces TypeScript
│   ├── layout/          # componente de layout base para estructura común de la app
│   ├── pages/           # pantallas o vistas principales
│   ├── hooks/           # hooks personalizados para lógica reutilizable
│   ├── services/        # llamadas a APIs y lógica de datos
│   ├── utils/           # funciones auxiliares usadas en varios lugares
│   ├── typings/         # tipos TypeScript personalizados
│   ├── styles/          # estilos globales adicionales
│   ├── App.css          # estilos del componente raíz App
│   ├── App.tsx          # componente principal de la aplicación
│   ├── index.css        # estilos globales de la app
│   └── main.tsx         # punto de entrada de React
├── index.html           # HTML base que carga la app
```
